Чтобы создать репозиторий, нужно прописать команду:



git init — создание пустой репозитории



Для современного состояния репозитория используется:



git status



Он показывает:



Какие файлы были изменены,

Что добавлено в коммите,

Что предстоит в коммите

Перенос строки в Gite — два пробела в текущей строке и Enter . Увы :(



Чтобы сохранить файл и добавить изменения в отслеживание, используйте:



git add filename — добавить конкретный файл с именем имя файла в отслеживание

git add . - добавить все изменения во все файлы (все файлы в репозиториях, которые менялись)



Фиксирует изменения (создает коммит):



git commit -m "Описание изменений"



Показывает историю коммитов в текущей ветке:



git log



Добавление пустого файла в проект:



touch имя\_файла
